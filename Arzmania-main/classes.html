Système de Classes d'Arzmania

Les classes définissent les compétences et capacités de votre personnage dans le monde d'Arzmania. Une fois choisie, votre classe ne pourra pas être modifiée durant le RP, alors choisissez avec soin ! Le système est basé sur celui de Donjons & Dragons, apportant une richesse de gameplay unique.

💡 3 types de classes disponibles

🔥

Classes Avancées

Les classes les plus puissantes, nécessitant une grande connaissance et maîtrise. Ces classes offrent des capacités exceptionnelles et demandent une expérience approfondie pour être jouées correctement.

🛠️

Classes à Compétence

Ces classes exigent des compétences spécifiques, mais pas nécessairement un don particulier. Elles sont idéales pour les joueurs cherchant à se spécialiser dans des domaines particuliers.

⚔️

Classes Basiques

Accessible à tous, ces classes n'ont pas les compétences les plus puissantes mais jouent un rôle crucial dans Arzmania. Elles offrent une base solide pour commencer l'aventure et évoluer avec le temps.

📜 Les catégories de classes d'Arzmania

⚔️ Combat

Les classes de combat se concentrent sur la maîtrise des armes et la puissance physique. Qu'il s'agisse de défense ou d'attaque, ces classes sont les piliers des batailles en première ligne.

🔮 Magie

Les classes magiques manipulent les forces surnaturelles pour attaquer, défendre ou soutenir leurs alliés. Ces utilisateurs de la magie puisent leur pouvoir dans des arcanes, la nature, ou les éléments.

🏹 Agilité

Ces classes reposent sur la furtivité, la rapidité et la ruse. Experts des embuscades, du vol, et de la dissimulation, ils agissent dans l'ombre et frappent vite et fort avant de disparaître.

🔰 Système de Choix des Classes

Le choix de votre classe principale se fait parmi toutes les classes disponibles, peu importe leur type. Cependant, le type de votre classe principale influencera le type de votre sous-classe :

🔥 Classe principale Avancée

La sous-classe doit être de type Basique, pour équilibrer la puissance de votre personnage.

Exemple : Paladin / Voleur

🛠️ Classe principale à Compétence

La sous-classe doit être de type à Compétence, afin d'apporter de la polyvalence sans surcharger en compétences.

Exemple : Illusionniste / Soldat

⚔️ Classe principale Basique

La sous-classe doit être de type Avancée, offrant une spécialisation ou puissance supplémentaire à votre personnage.

Exemple : Guerrier / Navigateur

Information supplémentaire
