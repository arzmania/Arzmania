Stryx

Souverains des Cieux...

Description

Les Stryx sont une race pacifique vivant dans les cieux dans le royaume de Mesthoman.
Ces êtres vivants sont dotés de grandes ailes leurs permettant de voler dans les airs.

Attributs de Race

  • Capables de voler.
  • Une vision surdéveloppée.

Histoire

Le peuple Stryx est resté en retrait durant la grande guerre, ils souhaitaient garder la paix éternelle. Depuis, les autres peuples trouvent que le peuple Stryx est un peu égoïste mais qu' il n'y a pas de discriminations. Ce peuple est ainsi un peuple entièrement démocratique essayant de baigner dans un semblant de paix que nul ne pourra ébranler.

Espérance de Vie

300 ans
